### ElectroZone – E-Commerce Electronics Website
ElectroZone is a modern and responsive e-commerce website for electronics products.
The project focuses on UI/UX design, product browsing experience, and interactive cart functionality.
###  Tech Stack
HTML5
CSS3
Bootstrap
JavaScript (DOM & LocalStorage)
Font Awesome Icons
### Home Page
Responsive modern layout

Bootstrap grid system

Product sections divided by category

Product cards dynamically loaded from products.json

Discount badge & old price logic

Star rating style

Add to Cart & Favourite icons
### Shopping Cart System
Cart data is saved using LocalStorage so it remains after page reload.

Includes:

Add product to cart

Prevent duplicate items — increase quantity instead

Quantity control (+ / −)

Remove product from cart

Update total price automatically

Cart counter in header

Sync button state across all product sections

Slide-in cart drawer UI
### Dynamic Product Rendering
products.json
Displayed based on:

Category type

Discount availability

Smart filtering logic
